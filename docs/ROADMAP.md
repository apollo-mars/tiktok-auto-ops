# Roadmap

## MVP (14 days)
### Day 1–2: Repo bootstrap
- Docker compose: Postgres + NestJS API + Python worker + Next.js console
- Env management and dev scripts

### Day 3–5: Product Hub + Asset Library
- DB migrations for products/product_assets
- CRUD APIs + basic UI forms

### Day 6–7: Script Engine
- Script templates + variant generation (vi)

### Day 8–10: Render Worker (Python)
- 3 render templates: compare/pain/unboxing
- Subtitles + TTS integration stub

### Day 11–12: Scheduler + Human Publish Gate
- publish_tasks queue + approval UI

### Day 13–14: Analytics feedback loop
- video_metrics ingestion (manual entry MVP)
- simple rules-based post-mortem suggestions

## Scale-up (after MVP)
- Multi-account matrix rules
- Automated metrics ingestion
- Stronger compliance checks and asset rights audit
- A/B testing dashboard
